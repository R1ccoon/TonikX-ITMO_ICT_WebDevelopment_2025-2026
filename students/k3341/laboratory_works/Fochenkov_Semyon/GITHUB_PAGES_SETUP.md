# Настройка GitHub Pages

Если на GitHub Pages отображается README.md вместо собранного сайта, выполните следующие шаги:

1. Перейдите в настройки репозитория: **Settings** → **Pages**

2. В разделе **Source** выберите:
   - **Deploy from a branch**
   - **Branch:** `gh-pages`
   - **Folder:** `/ (root)`

3. Сохраните изменения

4. Дождитесь завершения деплоя (обычно 1-2 минуты)

5. Обновите страницу GitHub Pages

После этого должен отображаться собранный сайт MkDocs, а не README.md.
